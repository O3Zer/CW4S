1.	Техническое задание на выполнение курсовой работы

Создать класс «Двунаправленный список», элементами которого являются экземпляры класса «Товар».
Класс «Товар» включает поля: номер магазина, название товара, цена товара.
Определить название магазина, в котором заданный товар самый дешевый.

Описание структуры входных и выходных данных

Для выбора пункта меню, необходимо ввести номер пункта меню и подтвердить ввод клавишей Enter. 
При выборе пункта меню 1 необходимо выбрать пункт подменю, при выборе пункта 2 программа возвращается в главное меню, а при выборе пункта 1 необходимо последовательно ввести данные о новом товаре, который будет записан в список (здесь и далее будет пониматься список товаров, введенных через первый пункт меню):
•	Номер магазина из предоставленного программой списка. Программа выдаст ошибку, если номер магазина не соответствует ни одному из предложенных программой;
•	Название товара – любая произвольная строка;
•	Цена– любое целочисленное число;
Выходными данными для этого пункта меню будет подтверждение добавления товара.
При выборе пункта меню 2 необходимо последовательно ввести название продукта. Выходными данными будет название магазина, в котором этот товар самый дешевый, и его цена. Программа выдаст ошибку, если ни один товар не записан в список.
При выборе пункта меню 3 не требуется ввод входных данных. Выходными данными будут все элементы списка. 
При выборе пункта меню 4 необходимо ввести номер магазина и название продукта, который пользователь хочет удалить. Программа выдаст ошибку, если ни один товар не записан в список, и, если программа не нашла товар, название которого ввел пользователь.
При выборе пункта меню 5 необходимо выбрать номер магазина и ввести название продукта, который пользователь хочет изменить, а затем выбрать новый номер магазина, название товара и его цену. Программа выдаст ошибку, если ни один товар не записан в список, и, если программа не нашла товар, название которого ввел пользователь.
При выборе пункта меню 6 программа завершает свою работу.
При вводе отличного символа, программа сообщает пользователю о некорректных введенных данных и продолжает работать.
